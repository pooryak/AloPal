title: File Structures
---

Each page has it's own folder in pages directory.in each component,widget or pages there are three different file.

## index.js

these files duty is to export their sibling component or folders.

## component.js

these files duty is to render component content(except for pages routes).

## style.js

these files duty is to provide component's styles(except for pages routes which the styles will be used in index.js itself).
