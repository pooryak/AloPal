Each page has it's own folder in pages directory.in each component,widget or pages there are three different file.

## next.config.js

a url loader injected into the webpack rules for handler and the bundler analyzer configured for analyzing the build files(for running try this : yarn build-analyze).

## jsconfig.js

Base url defined in this file for absolute imports in project.

## jest.config.js and jest.setup.js

Jest settings for testing environment

## constant.js

It consists of basic settings for api call in project (baseUrl,time-out,content-type,Accept)

## .eslintrc.js

eslint settings (airbnb)

## .babelrc

babel settings
